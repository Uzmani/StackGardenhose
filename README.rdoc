==Readme
This is our (Rhonda, George, Richie, me) stack overflow clone project 

Right now,
  - user can create an account 
  - a user can login (using has_secure_password)
  - post a question
  - answer a question
  - user can view questions and answers (logged in or not)
  - user can log out (clear session)
  - user can vote on questions 
  
  
Contains Unit and controller tests with RSPEC
